# Multi-Agent Ghost System - Requirements

## Overview
A system of 6 specialized AI ghost agents, each with unique expertise, personality, and voice, working together to analyze legacy code.

## User Stories

### US-1: Ghost Selection
**As a** developer  
**I want** to choose which ghost to summon  
**So that** I get specialized help for my specific problem

**Acceptance Criteria:**
- [x] Display 6 ghost personas with names, roles, and specialties
- [x] Show death cause for each ghost (adds personality)
- [x] Visual cards with emoji icons
- [x] Hover effects reveal ghost quotes
- [x] Selected ghost highlighted with glow effect

### US-2: Specialized Responses
**As a** developer  
**I want** each ghost to respond based on their expertise  
**So that** I get relevant, specialized advice

**Acceptance Criteria:**
- [x] Adrian focuses on architecture and patterns
- [x] Marcus emphasizes security vulnerabilities
- [x] Beatrice talks about infrastructure and deployment
- [x] Elena discusses data and ML aspects
- [x] Carlos references legacy systems and history
- [x] Sofia hunts for bugs and testing gaps

### US-3: Unique Personalities
**As a** developer  
**I want** each ghost to have a distinct personality  
**So that** interactions feel unique and entertaining

**Acceptance Criteria:**
- [x] Each ghost references their death cause
- [x] Sarcastic comments match their specialty
- [x] Defensive about code they "wrote"
- [x] Dark humor about being dead

### US-4: Voice Differentiation
**As a** developer  
**I want** male and female ghosts to have different voices  
**So that** I can distinguish them in voice chat

**Acceptance Criteria:**
- [x] Male ghosts (Adrian, Marcus, Carlos) use "Puck" voice
- [x] Female ghosts (Beatrice, Elena, Sofia) use "Aoede" voice
- [x] Voice changes when switching personas

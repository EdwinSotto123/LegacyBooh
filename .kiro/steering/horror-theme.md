# Horror Theme Guidelines

## Visual Style
- **Background**: Deep blacks (#0a0505, #0c0c0c)
- **Primary**: Blood red (#ef4444, #dc2626)
- **Secondary**: Amber/gold (#f59e0b, #d97706)
- **Accent**: Purple for mystical elements (#7c3aed)

## Typography
- `font-creep`: Horror display font for titles
- `font-old`: Old English style for dramatic text
- `font-mono`/`font-code`: Monospace for code

## UI Elements
- Glowing borders with box-shadow
- Scanline effects on code editor
- Pulsing animations for alerts
- Shake animations for errors

## Sound Design
- `/assets/grito.mp3` - Scream for critical errors
- `/assets/disparo.mp3` - Exorcism sound effect
- `/assets/back_ground.mp3` - Ambient horror music
- `/assets/bug.mp3` - Bug detection sound
- `/assets/pisada.mp3` - Footstep for navigation

## Ghost Personality
- Sarcastic but helpful
- References to being dead: "From beyond the grave..."
- Defensive of their legacy code
- Dark humor about technical debt
